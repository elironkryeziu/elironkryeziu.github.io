# elironkryeziu.github.io

Personal resume site for **Eliron Kryeziu** — Back-end Developer.

🔗 Live: https://elironkryeziu.github.io

## About

The site is a single, self-contained `index.html` (designed with Claude). All
styling and assets are inlined and unpacked client-side, so there's no build
step and no external dependencies — GitHub Pages serves the file as-is.

## Contents

| File | Purpose |
|------|---------|
| `index.html` | The resume / landing page |

## Deploying changes

GitHub Pages publishes automatically from the `main` branch (root). Push to
`main` and the site rebuilds within a minute or two.
